-------------------------------------------------------------------------------
1.图片OCR和NLP
-------------------------------------------------------------------------------
    发送：
    Request-Method:   POST
	Request-Type:     Resolve-Image
	Request-Property: application/octet-stream(字节流-ByteArrayOutputStream)
	发送内容：图片的字节流
-------------------------------------------------------------------------------
	接收JSON格式:
	{   "memoId":"00000120160209222735348",
        "topic":"happy new year",
        "time":"2016-02-07 15:32",
	    "content":"新年快乐"
    }
-------------------------------------------------------------------------------
2.列表请求
-------------------------------------------------------------------------------
    发送：
	Request-Method:   POST
	Request-Type:     Get-Memo-List
	Request-Property: text/plain(纯文本)
	发送内容：{"userid":"000001"}
-------------------------------------------------------------------------------
	接收JSON格式:
	{
		"memo":
	    [
		    {
			    "memoId":"00000120160209222735348",
			    "topic":"happy new year",
                "time":"2016-02-07 15:32",
	            "content":"新年快乐"
			},
			{
			    "memoId":"00000120160209222735348",
			    "topic":"happy new year",
                "time":"2016-02-08 15:32",
	            "content":"新年快乐"
			}
		]
	}
-------------------------------------------------------------------------------
3.内容删除
-------------------------------------------------------------------------------
    发送：
	Request-Method:   POST
	Request-Type:     Delete-Memo
	Request-Property: application/json(JSON格式)
	发送内容：删除的Memo id
	{
	    "memoId":"00000120160209222735348"
    }
-------------------------------------------------------------------------------
    接收RequestCode：200为成功，其他为失败
-------------------------------------------------------------------------------
4.内容修改
-------------------------------------------------------------------------------
    发送；
	Request-Method:   POST
	Request-Type:     Modify-Memo
	Request-Property: application/json(JSON格式)
	发送内容：修改的Memo内容和用户名
	{
	    "memoId":"00000120160209222735348",
        "topic":"happy new year",
        "time":"2016-02-07 15:32",
	    "content":"新年快乐"
    }
-------------------------------------------------------------------------------
    接收RequestCode：200为成功，其他为失败
-------------------------------------------------------------------------------
5.登录
-------------------------------------------------------------------------------
    发送；
	Request-Method:   POST
	Request-Type:     Sign-In
	Request-Property: application/json(JSON格式)
	发送内容：用户名和密码
    {
	    "userName":"Alan"
        "password":"123456"
    }
	加密算法以后再补充
-------------------------------------------------------------------------------
    接收RequestCode：200为成功，其他为失败
	接受JSON：
	{
	    "userId":"000001"
	}
-------------------------------------------------------------------------------
6.获取头像
-------------------------------------------------------------------------------
    发送；
	Request-Method:   POST
	Request-Type:     Get-Logo
	Request-Property: application/json(JSON格式)
	发送内容：用户名和密码
    {
	    "userId":"000001"
    }
-------------------------------------------------------------------------------
    接收RequestCode：200为成功，其他为失败
	接受图片字节流
-------------------------------------------------------------------------------
7.获取签名
-------------------------------------------------------------------------------
    发送；
	Request-Method:   POST
	Request-Type:     Get-Signature
	Request-Property: application/json(JSON格式)
	发送内容：用户名和密码
    {
	    "userid":"000001"
    }
-------------------------------------------------------------------------------
