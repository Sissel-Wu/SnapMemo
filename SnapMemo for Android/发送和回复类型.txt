-------------------------------------------------------------------------------
1.图片OCR和NLP
-------------------------------------------------------------------------------
    发送：
    Request-Method:   POST
	Request-Type:     Resolve-Image
	Request-Property: application/octet-stream(字节流-ByteArrayOutputStream)
-------------------------------------------------------------------------------
	接收JSON格式:
	{
        "topic":"happy new year"
        "date":"2016-02-07"
	    "content":"新年快乐"
    }
-------------------------------------------------------------------------------
2.列表请求
-------------------------------------------------------------------------------
    发送：
	Request-Method:   POST
	Request-Type:     Get-List
	Request-Property: text/plain(纯文本)
-------------------------------------------------------------------------------
	接收JSON格式:
	{
		"memo":
	    [
		    {
			    "topic":"happy new year"
                "date":"2016-02-07"
	            "content":"新年快乐"
			},
			{
			    "topic":"happy new year"
                "date":"2016-02-08"
	            "content":"新年快乐"
			}
		]
	}
-------------------------------------------------------------------------------
3.内容删除
-------------------------------------------------------------------------------
    发送：
	Request-Method:   POST
	Request-Type:     Delete-Content
	Request-Property: application/json(JSON格式)
	{
	    "userName":"Alan"
        "topic":"happy new year"
        "date":"2016-02-07"
	    "content":"新年快乐"
    }
-------------------------------------------------------------------------------
    接收RequestCode：200为成功，其他为失败
-------------------------------------------------------------------------------
4.内容修改
-------------------------------------------------------------------------------
    发送；
	Request-Method:   POST
	Request-Type:     Modify-Content
	Request-Property: application/json(JSON格式)
	{
	    "userName":"Alan"
        "topic":"happy new year"
        "date":"2016-02-07"
	    "content":"新年快乐"
    }
-------------------------------------------------------------------------------
    接收RequestCode：200为成功，其他为失败
-------------------------------------------------------------------------------
5.登录
-------------------------------------------------------------------------------
    发送；
	Request-Method:   POST
	Request-Type:     Sign-In
	Request-Property: application/json(JSON格式)
    {
	    "userName":"Alan"
        "password":"123456"
    }
	加密算法以后再补充
-------------------------------------------------------------------------------
    接收RequestCode：200为成功，其他为失败
-------------------------------------------------------------------------------
